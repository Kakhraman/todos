# Minimalist Todo SPA

A modern, fast, and minimalist Single Page Application (SPA) for managing todos. Built with Node.js, Express, and a sleek Bootstrap v5 interface.

## ✨ Features

- **Single Page Interface:** Seamless user experience without page reloads.
- **Full CRUD Support:** Add, View, Edit, and Soft-delete tasks.
- **Task Specs:** Click any task to view detailed specifications (ID, Status, Created/Modified timestamps).
- **Toggle Completion:** One-tap circular radio buttons to mark tasks as complete or active.
- **Advanced Filtering:** Switch between "All", "Active", and "Completed" views.
- **Real-time Search:** Instantly find tasks by title or description.
- **Pagination:** Clean, server-side pagination (10 items per page).
- **Persistent Storage:** Data is stored in a local `todo.json` file.
- **Keyboard Shortcut:** Quick task entry using the `Enter` key.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kakhraman/todos.git
   cd todos
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm start
   ```

4. Open your browser and visit:
   `http://localhost:3000`

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Frontend:** Vanilla JS, Bootstrap v5, Bootstrap Icons
- **Storage:** JSON File System
